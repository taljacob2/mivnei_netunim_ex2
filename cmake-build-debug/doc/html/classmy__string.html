Attention
  • this implementation enforces the string to ignore the following white-spaces:
    ' ', '\t', '\n'
    . and receives only non-white-spaces.
  • the method does not include the last
    '\n'
    in the string and does not leave it in the is istream. instead, the method reads it but ignores it.
