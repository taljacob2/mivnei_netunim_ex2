◆ fixHeap()

template<typename K >
void MinHeap< K >::fixHeap ( int  indexToFixFrom )
inlineoverridevirtual

This method handles a heap that is valid from the root downwards until the indexToFixFrom, which from there and on downwards, the heap is invalid - means: that the node in the indexToFixFrom is no smaller than both of its children.

Parameters
indexToFixFrom the method fixes the heap from this index downwards until the leaves of the heap.
Note
this method will continue to run until the root is no longer smaller than both of its children, or when the root is a leaf.
Attention
there is no use to give indexToFixFrom that is larger than (logicalSize / 2), because indexes larger than (logicalSize / 2) point to leaf nodes, thus the method will have no effect, as explained earlier.

Implements MinHeapADT< K >.
