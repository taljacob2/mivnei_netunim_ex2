int  indexToElement1,
int  indexToElement2

inlinestatic

Compares two elements pointed out from two pointers, by their given indexes, and returns the index to the pointer that points to the larger element out of them.

The method ensures safety:

  • The method makes sure to ensure that the indexToElement1 and indexToElement2 provided are not out of range.
  • The method makes sure to ensure that the elements are not nullptr, and if they are nullptr, the method knows how to handle them.
    Note
    if element is a nullptr, prioritize to return the index of the other element given. If both element are nullptr, return the index of the first element.
    Attention
    if indexToElement2 is out fo range, but indexToElement1 is not out of range, the method returns the indexToElement1.
    Parameters
    array an array of pointers to elements.
    indexToElement1 index of pointer to first element to compare.
    indexToElement2 index of pointer to second element to compare.
    Returns
    the index of the pointer that points to the larger element.
    Exceptions
    std::out_of_range in case both of the indexes indexToElement1 and indexToElement2 provided are out of range.
