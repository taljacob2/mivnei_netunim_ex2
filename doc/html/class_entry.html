Detailed Description

template<typename K, typename V>
class Entry< K, V >

This class represents a general entry, which is composed from a key and a value.

Entries are compared to each other, by the comparable key field located in each Entry element.

Attention
the key must be comparable.
Template Parameters
K the type of key in the entry.
V the type of value in the entry.
