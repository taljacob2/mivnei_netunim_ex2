inlinestaticprivate

This method gets an input from an input-stream is, and after that, checks whether the input gotten is legal by checking it with a specific function: checkingFunction. In case the input was found as legal, then we manipulate the checked input with the returnFunction, and its return value is set to be this method's return value.

Template Parameters
Tis the type of the value to get.
Parameters
isis the input-stream to get the value from.
checkingFunctionis the checking function that checks that the input gotten is legal.
returnFunctionis the function that is being invoked finally in its method. Its return value is the final return value of this method.
Returns
the checked input gotten.

this method is generic, and has a constant template.

Exceptions
std::runtime_errorif the input is in an illegal format.
