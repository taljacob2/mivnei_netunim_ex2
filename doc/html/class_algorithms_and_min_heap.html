This method divides a given array to k smaller arrays, and invokes the forEachSmallArrayFunction function for each of the divided smaller arrays.

Note
The method divides the given array such that the sizes of the smaller arrays are spread as equally as possible.
Template Parameters
K the type of elements in the array given. this would serve as the key type of the elements.
Parameters
array the array to divide to k smaller arrays.
size the size of the array given.
k the division parameter.
forEachSmallArrayFunction this function is being invoked for-each smaller array.

inlinestatic

This algorithm sorts the array given, by dividing it to k smaller arrays. Sorting each of them. And finally merging them together.

Note
in case k is smaller than the size of the array given, then the method invokes my_algorithms::quickSort(K *, int).
In case k equals to 2, this algorithm is actually the known MergeSort algorithm.
Template Parameters
K the type of the elements in the array given.
Parameters
array the array to sort.
size the size of the array to sort.
k the division parameter, when sorting the array.
See also
my_algorithms::quickSort(K *, int)

inlinestaticprivate

This private method is invoked only when we have removed an element from a small-array. The method ensures to step-ahead the location of the according small-array location, and to decrease by 1 the size of the according small-array size.

Warning
this method does not check that the smallArraySizes[deletedElement->getValue()] is a number greater than 0. Thus, you must use this method in a parentheses of:
if ((smallArraySizes[deletedElement->getValue()]) > 0) {
// call this method...
}
Template Parameters
K the key of the element.
Parameters
smallArrayLocations the array that stores the starting locations of all the small-arrays.
smallArraySizes the array that stores the sizes of all the small-arrays.
deletedElement the element that has been removed from a generic small-array.
