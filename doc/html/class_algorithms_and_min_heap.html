This method divides a given array to k smaller arrays, and invokes the forEachSmallArrayFunction function for each of the divided smaller arrays.

Attention
this method creates k small arrays that their starting location is stored in the smallArrayLocations array. So, after the use of this method, the user must delete each small array with a loop:
// Delete each `small array`.
for (int i = 0; i < k; i++) { delete[] smallArrayLocations[i]; }
Note
The method divides the given array such that the sizes of the smaller arrays are spread as equally as possible.

smallArraySizes is an array that stores the size of each small array.

minHeap is the Minimum-Heap that has been pre-initialized to the physicalSize of k. And in this method, on each iteration, right after the forEachSmallArrayFunction is being invoked, this Minimum-Heap is inserting the first element in each small array to it.

const unsigned int  size,
unsigned int  k 
)
inlinestatic

This algorithm sorts the array given, by dividing it to k smaller arrays. Then for each small array:

  • Sort the array.
  • insert the first element in the array to the Minimum-Heap provided.

Then, for size number of times:

  • Invoke the MinHeap.deleteMin() method to get the smallest element in the MinHeap, and insert it to the resultArray.
  • Determine the specific small array from which that element came from, and extract another the next smallest element in it to the MinHeap, instead of the removed element.

The method handles 3 arrays:

  • (K *) smallArrayLocations - is an array that stores the starting location of each small array.
  • (K **) changeableSmallArrayLocations - is the same as smallArrayLocations, but on each remove of a first element in a small array, set the location of the according small array to be 1 location ahead.
  • (int *) smallArraySizes - is an array that stores the size of each small array. The method creates a MinHeap: This MinHeap's elements are pointers to Entries such that each Entry is composed of a key and a value. Itskey, stands for the smallest element in eachsmall array- and itsvaluestands for an *int* number that determines the specificsmall array` from which that element came from.
Note
in case k is smaller than the size of the array given, then the method invokes my_algorithms::quickSort(K *, int).
in case k is less than 2, the method sets k to be 2, to prevent an error.
In case k equals to 2, this algorithm is actually the known MergeSort algorithm.

K the type of the elements in the array given.
Parameters
array the array to sort. Must be dynamically allocated.
size the size of the array to sort.
k the division parameter, when sorting the array.

This private method is invoked only when we have removed an element from a small-array. The method ensures to step-ahead the location of the according small-array location, and to decrease by 1 the size of the according small-array size.

Warning
this method does not check that the smallArraySizes[deletedElement->getValue()] is a number greater than 0. Thus, you must use this method in a parentheses of:
if ((smallArraySizes[deletedElement->getValue()]) > 0) {
// call this method...
}
Template Parameters
K the key of the element.
Parameters
changeableSmallArrayLocations the array that stores the current starting locations of all the small-arrays.
smallArraySizes the array that stores the sizes of all the small-arrays.
deletedElement the element that has been removed from a generic small-array.
