Detailed Description

template<typename K, typename V>
class MinHeapADT< K, V >

This class represents an abstract Minimum-Heap, which its elements are Entries that are composed by a key and a value.

The heap compares its elements to each other, by the comparable key field located in each Entry element.*

Template Parameters
K the type of key in the entry.
V the type of value in the entry.
Note
The terms element, node and 'entry' are synonyms.
