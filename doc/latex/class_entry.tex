\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename K, typename V$>$\newline
class Entry$<$ K, V $>$}

This class represents a general {\itshape entry}, which is composed from a {\itshape key} and a {\itshape value}. 

{\ttfamily Entries} are compared to each other, by the comparable {\ttfamily key} field located in each {\ttfamily \mbox{\hyperlink{class_entry}{Entry}}} element. \begin{DoxyAttention}{Attention}
the {\ttfamily key} {\bfseries{must}} be {\ttfamily comparable}. 
\end{DoxyAttention}
