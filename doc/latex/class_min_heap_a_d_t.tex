\subsubsection*{template$<$typename K, typename V$>$\newline
class Min\+Heap\+ADT$<$ K, V $>$}

This class represents an {\itshape abstract} {\bfseries{Minimum-\/\+Heap}}, which its elements are {\itshape Entries} that are composed by a {\itshape key} and a {\itshape value}. 

The heap compares its elements to each other, by the comparable {\ttfamily key} field located in each {\ttfamily \mbox{\hyperlink{class_entry}{Entry}}} element.$\ast$
