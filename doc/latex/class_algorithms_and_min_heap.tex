This algorithm sorts the {\ttfamily array} given, by dividing it to {\ttfamily k} {\ttfamily smaller arrays}. Then for each {\ttfamily small array}\+: 

\begin{DoxyItemize}
\item Sort the array. \item {\ttfamily insert} the {\ttfamily first} element in the array to the {\ttfamily Minimum-\/\+Heap} provided.\end{DoxyItemize}
Then, for {\ttfamily size} number of times\+: \begin{DoxyItemize}
\item Invoke the {\ttfamily \mbox{\hyperlink{class_min_heap_a6f4be46b00c888580c5f3afe41865930}{Min\+Heap.\+delete\+Min()}}} method to get the {\ttfamily smallest} element in the \mbox{\hyperlink{class_min_heap}{Min\+Heap}}, and insert it to the {\ttfamily result\+Array}. \item Determine the specific {\ttfamily small array} from which that element came from, and extract another the next {\ttfamily smallest} element in it to the {\ttfamily \mbox{\hyperlink{class_min_heap}{Min\+Heap}}}, instead of the removed element.\end{DoxyItemize}
The method handles 3 arrays\+: \begin{DoxyItemize}
\item (K $\ast$) small\+Array\+Locations -\/ is an array that stores the {\ttfamily starting} location of each {\ttfamily small array}. \item (K $\ast$$\ast$) changeable\+Small\+Array\+Locations -\/ is the same as {\ttfamily small\+Array\+Locations}, but on each {\ttfamily remove} of a {\ttfamily first} element in a {\ttfamily small array}, set the location of the according {\ttfamily small array} to be {\ttfamily 1} location ahead. \item (int $\ast$) small\+Array\+Sizes -\/ is an array that stores the {\ttfamily size} of each {\ttfamily small array}. The method creates a {\ttfamily \mbox{\hyperlink{class_min_heap}{Min\+Heap}}}\+: This {\ttfamily \mbox{\hyperlink{class_min_heap}{Min\+Heap}}}\textquotesingle{}s elements are pointers to {\ttfamily Entries} such that each {\ttfamily \mbox{\hyperlink{class_entry}{Entry}}} is composed of a {\ttfamily key} and a {\ttfamily value. Its}key{\ttfamily , stands for the smallest element in each}small array{\ttfamily -\/ and its}value{\ttfamily stands for an $\ast$int$\ast$ number that determines the specific}small array\`{} from which that element came from.\end{DoxyItemize}
\begin{DoxyNote}{Note}
in case {\ttfamily k} is smaller than the {\ttfamily size} of the {\ttfamily array} given, then the method invokes \mbox{\hyperlink{}{my\+\_\+algorithms\+::quick\+Sort(\+K $\ast$, int)}}. 

in case {\ttfamily k} is less than {\ttfamily 2}, the method sets {\ttfamily k} to be {\ttfamily 2}, to prevent an error. 

In case {\ttfamily k} equals to {\ttfamily 2}, this algorithm is actually the known {\ttfamily Merge\+Sort} algorithm. 
\end{DoxyNote}
